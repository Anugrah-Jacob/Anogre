# Cooking App Design System

## Colors
- Primary: #FF6B00 (Orange)
- Secondary: #000000 (Black)
- Background: #FFFFFF (White)
- Text Primary: #111111
- Text Secondary: #666666

## Typography
- Heading: Bold, large, modern sans-serif
- Body: Regular, clean sans-serif
- Button: Medium weight

## Spacing
- Small: 8px
- Medium: 16px
- Large: 24px

## Border Radius
- Buttons: 12px
- Cards: 16px

## Components
### Button
- Rounded corners
- Orange background
- White text

### Card
- White background
- Soft shadow
- Rounded corners

## Style
- Clean, modern, slightly playful
- Food-focused UI
- Use illustrations where possible
